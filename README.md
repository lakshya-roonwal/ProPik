## Run Locally

Clone the project

```bash
  git clone https://github.com/lakshya-roonwal/ProPik.git
```

Go to the project directory

```bash
  cd ProPik
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
